## 적용법
리소스팩으로 불러와 주시면 됩니다.  
`설정`에서 `리소스팩`으로 들어간 후  
팩 폴더에 리소스팩 파일을 집어넣어 가져오거나  
화면 밖에서 드래그 앤 드랍으로 가져오실 수 있습니다.
